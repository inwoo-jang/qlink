# 유저플로우 설계 스킬

## 핵심 플로우 카탈로그

### 1. URL 저장 플로우
```
[홈] →(+ 버튼)→ [추가 시트]
   ↓
   ├─ URL 붙여넣기 (클립보드 자동인식)
   └─ QR 스캔 → [카메라 뷰] → 인식 성공
   ↓
[메타 추출 + AI 요약 진행 표시]
   ↓
[홈에 새 카드 노출 + 토스트 "저장됨"]
```

### 2. 검색 플로우
```
[하단 탭 → 검색] → [검색 입력]
   ↓
   ├─ 최근 검색어 / 추천 태그
   └─ 키워드 입력 → 디바운스 300ms → 결과 갱신
       ├─ 제목 매칭
       ├─ 요약 매칭 (하이라이트)
       └─ 태그 매칭
```

### 3. 폴더링 플로우
```
[홈에서 카드 길게 누르기] → [선택 모드]
   ↓
[다중 선택] → [폴더로 이동] → [폴더 선택 시트]
   ↓
[카드 폴더로 이동 애니메이션 + 토스트]
```

### 4. 공유/열기 플로우
```
[링크 카드 탭] → [상세 화면]
   ↓
   ├─ 원본 열기 (in-app browser 또는 외부)
   ├─ 공유 (Web Share API → Kakao/SMS/...)
   └─ 알림 설정 → [날짜·시간 선택]
```

### 5. 알림 수신 플로우
```
[설정한 시간 도달] → [Web Push 도착]
   ↓
[탭 → 큐링크 열림] → [해당 링크 상세]
```

## 작성 가이드
- 각 단계는 화면 상태 또는 사용자 액션으로 표기
- 분기점은 들여쓰기 또는 화살표로 명확히
- 에러 분기(권한 거절, 네트워크 실패 등)도 별도 섹션으로 작성
