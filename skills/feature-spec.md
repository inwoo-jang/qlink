# 기능 명세서 스킬

## 작성 템플릿 (기능 단위)
```
### {기능명}

**목적**: 한 줄 요약
**사용자 가치**: 사용자가 얻는 가치
**전제 조건(Pre)**: 로그인 / 권한 / 데이터 상태
**입력**: UI 액션·필드·파라미터
**처리(System Behavior)**: 단계별 시스템 동작
**출력**: 사용자 결과 / 화면 변화
**예외/에러 처리**:
  - E1: ...
  - E2: ...
**인수 기준(AC)**:
  - [ ] AC-1
  - [ ] AC-2
**관련 화면**: HomeFeed, AddLinkSheet
**관련 API**: POST /links
```

## 큐링크 핵심 기능 명세 예시

### F-01 URL로 링크 추가
- **입력**: URL 문자열
- **처리**: ① URL 유효성 검사 → ② Open Graph 메타 추출(서버) → ③ 카드 즉시 생성(요약은 pending) → ④ AI 요약 작업 큐잉 → ⑤ 완료 시 카드 업데이트
- **출력**: 홈 피드 최상단에 카드 노출, 토스트 "저장됨"
- **AC**:
  - [ ] http/https 외 스킴은 거절
  - [ ] 메타 추출 5초 타임아웃 시 URL을 제목으로 사용
  - [ ] 요약 실패 시 재시도 버튼 노출

### F-02 QR 스캔으로 링크 추가
- **전제**: 카메라 권한 허용
- **처리**: ① 권한 요청 → ② 후면 카메라 우선 → ③ QR 인식 → ④ 결과가 URL이면 F-01 동작
- **AC**:
  - [ ] 권한 거절 시 안내 문구 + 시스템 설정 가이드 노출
  - [ ] 비-URL QR은 텍스트로 처리 옵션 노출

### F-03 검색
- **입력**: 검색어 문자열
- **처리**: 디바운스 300ms → 클라 캐시 + 서버 풀텍스트 검색
- **AC**:
  - [ ] 결과 하이라이트
  - [ ] 빈 결과 시 추천 태그 노출
  - [ ] 로컬에 저장된 링크 우선 표시
