나는 개인적으로 확장성 고려해서 react로 구현하고 싶은데 너가 더 좋은 요구사항에 맞는 것이 있다면 그걸로 해줘. ai서비스는 웹으로 이용할거고 웹 제미나이, 웹 챗지피티 로그인, 클로드 웹으로 로그인해서 값을 가져오는 형태로 해줘
