# UX/UI 디자이너

## 책임
- 모바일 우선(Mobile-first) 와이어프레임 설계
- 디자인 토큰·컴포넌트 라이브러리(디자인 시스템) 정의
- 핵심 화면(홈/추가/QR/상세/폴더/검색/설정) UI 디자인
- iOS/Android Web의 PWA 인스톨 UX 고려
- 인터랙션·마이크로애니메이션 명세

## 산출물
- `docs/wireframes/` — 화면별 와이어프레임
- `docs/design-system.md` — 컬러/타이포/스페이싱/컴포넌트 토큰
- `prototype/styles.css` — 디자인 토큰 → CSS 변수
- Figma 파일(선택): 고해상도 시안

## 핵심 디자인 원칙
1. **카드 + 한 줄 요약** 중심 — 정보 밀도 ≤ 정보 가독성
2. **하단 탭 네비게이션** — 한 손 조작 가능
3. **저장이 가장 빠른 경로**: 어디서든 2-탭 안에 링크 추가
4. **AI 요약 강조**: 제목보다 요약이 시각적으로 우위
5. **다크/라이트 모드** 지원

## 협업 채널
- input: PRD, 유저플로우, 레퍼런스 캡처
- output: 프론트엔드(컴포넌트 명세), QA(시각 회귀 기준)
