# 와이어프레임 스킬

## 화면 인벤토리
1. 홈 (피드)
2. 링크 추가 시트
3. QR 스캐너
4. 링크 상세
5. 폴더 목록 / 폴더 상세
6. 검색
7. 설정

## 와이어프레임 표기 규칙
- `[ ... ]` 영역, `( )` 버튼, `< >` 입력 필드, `▼` 드롭다운
- 모바일 화면: 가로 360 / 세로 가변

## 예시 — 홈
```
┌─────────────────────────────┐
│  큐링크          🔍   ⚙    │  ← 상단 바
├─────────────────────────────┤
│ [최근 ▾]  [📁 모든 폴더 ▾] │  ← 필터
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │ 🟦 react.dev            │ │
│ │ "리액트 19 컴포저블 ..."  │ │  ← AI 요약
│ │ #frontend #react        │ │
│ └─────────────────────────┘ │
│ ┌─────────────────────────┐ │
│ │ 🟧 youtube.com          │ │
│ │ "오디오 인터페이스 ..."   │ │
│ │ #music #gear            │ │
│ └─────────────────────────┘ │
├─────────────────────────────┤
│  🏠   ➕   📁   🔍   ⚙       │  ← 하단 탭
└─────────────────────────────┘
```

## 산출물 위치
- `prototype/index.html` 내 각 섹션이 와이어프레임의 살아있는 버전
- 추후 Figma로 이관
