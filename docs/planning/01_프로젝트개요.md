# 큐링크 (QLINK)

작성일: 2026-05-07
작성자: 인우

## 1. 프로젝트명

큐링크 (QLINK)

## 2. 한줄 설명

URL과 QR로 모은 링크를 AI가 한 줄 요약하고 폴더·태그로 분류해주는, 모바일·웹을 잇는 큐레이션 북마크 서비스.

## 3. 배경 및 목적

### 배경

링크를 받는 채널이 폭증했다. 단톡방·인스타DM·트위터·링크드인·뉴스레터·유튜브 알고리즘·슬랙 채널 — 하루 30~50개 링크가 흘러들어온다. 사용자는 "나중에 봐야지" 하고 어딘가에 저장하지만, 막상 다시 찾으려 하면 어디서 봤는지조차 헷갈린다. 기존 북마크 매니저는 PC·확장 프로그램 중심이고, 모바일에서 빠르게 캡처하고 의미를 부여하는 흐름이 약하다. 동시에 크롬 즐겨찾기는 폴더가 통제 없이 늘어나고, 계정이 바뀌거나 동기화가 끊기면 즐겨찾기 자체가 사라지거나 뒤섞인다.

QLINK는 **단일 PWA 코드베이스로 모바일·웹을 동시에 제공**하고, 계정 단위로 끊김 없이 동기화한다.

### 페인포인트

- "인스타에서 봤는지 네이버 카페에서 봤는지 헷갈려서 못 찾는 링크가 너무 많다"
- "단톡방에서 친구가 공유한 링크가 새 메시지로 묻혀서 끝없이 스크롤을 올려야 한다"
- "북마크에 저장해도 제목만 있고, 한 달 뒤엔 뭐였는지 기억이 안 난다"
- "유튜브 영상을 나중에 보려고 저장해두지만 결국 다시 안 본다"
- "친구들과 같은 주제 링크를 공유할 때 매번 단톡방에 다시 보내야 한다"
- "크롬 즐겨찾기 폴더가 통제 없이 끝도 없이 늘어나서, 정작 필요할 때 못 찾는다"
- "회사 PC와 집 PC, 그리고 휴대폰 — 크롬 계정이 다르거나 동기화가 끊기면 즐겨찾기가 사라지거나 다른 게 나타난다"
- "PC에서 본 링크를 모바일에서, 모바일에서 본 링크를 PC에서 이어서 보고 싶은데 늘 어딘가에 따로 저장된다"

### 시장 기회

기존 북마크/링크 정리 서비스는 두 갈래로 나뉜다.
- (a) PC·웹 클리퍼 중심 (Raindrop.io, Pocket) — 데스크톱 전제
- (b) 자동 정리 AI (mymind) — 월 $7~13 유료, 이미지 중심

QR 스캔 + 모바일·웹 동시 PWA + AI 한 줄 요약 + 공유 폴더 + 크롬 즐겨찾기 import를 한 자리에 묶은 한국어 친화 서비스는 비어 있다. 카톡 링크 공유 문화·QR 결제·자격증 강의 시청·다중 디바이스 사용 같은 한국 사용자 행동에 맞춘 진입점이 차별점.

### 차별화 포인트

- **QR 스캔을 1차 입력으로 전면화** — 분석한 5개 중 카메라 QR 입력을 메인에 둔 서비스 없음 (추정).
- **요약을 "읽기 전 필터"로 사용** — 저장 후 정리가 아닌, 저장 직전에 "이 링크 볼 가치 있나"를 판단하게 함.
- **외부 AI 계정(Gemini/ChatGPT/Claude) 세션 활용 구조** — 운영 비용 0원 지향. 다른 서비스는 자체 LLM 호출(원가 부담).
- **단일 PWA로 모바일·웹 동시 제공** — 설치 마찰 0. 한 계정으로 회사 PC ↔ 모바일 ↔ 집 PC 자연스럽게 이어짐. 크롬 즐겨찾기처럼 계정 동기화 깨질 위험 없음.
- **공유 폴더를 부수 기능이 아닌 1차 기능으로** — "단톡방에서 친구가 보낸 링크 묻혀서 못 찾는 문제"를 직접 해결.
- **크롬 즐겨찾기 HTML import** — 통제 잃은 즐겨찾기를 한 번에 가져와 AI 요약·폴더화로 다시 정돈.
