## 1. 사이트맵

```
QLINK PWA (모바일·웹 단일 코드베이스)
│
├── 인증 (미인증)
│   ├── #SPLASH                — 자동 토큰 검증·분기
│   ├── #AUTH-LOGIN            — 로그인 / "회원가입" 탭 / "둘러보기"(프로토타입)
│   ├── #AUTH-SIGNUP           — 가입 폼 (이메일·비밀번호·닉네임)
│   └── #AUTH-EMAIL-VERIFY     — 6자리 인증 코드
│
└── 메인 (인증 후)
    ├── 🏡 새 탭 (★ NEWTAB-001 — 시작 페이지 모드)
    │   └── #NEWTAB            — 큰 중앙 검색 + 오늘 할 일 + 최근 저장 + 자주 폴더 + 시간 인사
    │       — 브라우저 시작 페이지·새 탭으로 설정 가능. 설정에서 default 모드 토글
    │
    ├── 홈 (전체 보기)
    │   ├── #HOME              — 최신순 피드 (모바일 1열 / 웹 2~4열)
    │   ├── #HOME-EMPTY        — Empty State
    │   ├── #LINK-ADD          — 링크 추가 시트 (URL/QR/공유 탭)
    │   │   └── #QR-SCAN       — 카메라 QR 스캔 (모바일 전용)
    │   ├── #LINK-DETAIL       — 링크 상세 (모바일 시트 / 웹 사이드패널)
    │   └── #LINK-EDIT         — 제목·요약·태그 수정
    │
    ├── 폴더
    │   ├── #FOLDER-LIST       — 사용자 폴더 + 공유 폴더 섹션
    │   ├── #FOLDER-CREATE     — 폴더 생성 모달 (이모지+이름)
    │   ├── #FOLDER-DETAIL     — 폴더 내 링크 목록
    │   ├── #FOLDER-PICKER     — 폴더 선택 모달 (링크 이동 시)
    │   ├── #SHARE-INVITE      — 공유 링크 발송 (오너)
    │   ├── #SHARE-ACCEPT      — 초대 수락
    │   └── #FOLDER-MEMBERS    — 멤버 관리 (역할·제거·나가기)
    │
    ├── 할일 (탭)              — ★ Must 기능
    │   ├── #TODO-LIST         — 모든 할 일 평탄화 뷰 (필터 5종)
    │   ├── #TODO-CREATE       — 할 일 추가 시트 (제목·시간·반복)
    │   └── #TODO-EDIT         — 할 일 수정·삭제
    │
    ├── 검색
    │   ├── #SEARCH            — 키워드 입력 + 결과
    │   └── #SEARCH-EMPTY      — 결과 없음
    │
    ├── 설정
    │   ├── #SETTINGS          — 메뉴 진입점
    │   ├── #SETTINGS-PROFILE  — 닉네임·아바타
    │   ├── #SETTINGS-AI       — AI 제공자 + API 키
    │   ├── #SETTINGS-PASSWORD — 비밀번호 변경 (안내 배너)
    │   ├── #SETTINGS-THEME    — 라이트/다크 + 강조색 3종
    │   ├── #SETTINGS-NOTIFICATION — 모바일/PC 채널 토글
    │   ├── #SETTINGS-IMPORT   — 크롬 즐겨찾기 (웹 전용)
    │   │   ├── #IMPORT-UPLOAD
    │   │   ├── #IMPORT-PREVIEW
    │   │   └── #IMPORT-PROGRESS
    │   └── #SETTINGS-RESET    — 데이터 초기화 (모달)
    │
    └── 외부 진입
        ├── #SHARE-RECEIVE     — Web Share Target (외부 앱 → QLINK 공유)
        └── /invite/{token}    — 공유 폴더 초대 링크 클릭
```

총 30개 화면. (인증 4 + 홈/링크 7 + 폴더/공유 7 + 검색 2 + 설정 9 + 외부 2 — 일부 모달은 상위 화면의 변형으로 카운트)

## 2. 화면 카탈로그

### 2.1 인증

| 화면 ID | 목적 | 진입점 | 핵심 구성 | 관련 기능 |
|---|---|---|---|---|
| #SPLASH | 토큰 검증 분기 | 앱·웹 진입 | 로고, 로딩 인디케이터 | AUTH-004 |
| #AUTH-LOGIN | 로그인 + 가입·둘러보기 진입점 | #SPLASH 분기 | 이메일·비밀번호 폼, 회원가입 탭, "둘러보기"(프로토타입 한정) | AUTH-002, AUTH-003 |
| #AUTH-SIGNUP | 가입 폼 | #AUTH-LOGIN | 이메일·비밀번호·닉네임, 정책 안내 | AUTH-001 |
| #AUTH-EMAIL-VERIFY | 6자리 코드 검증 | 가입 직후 | 6자리 인풋, 재발송 버튼, 만료 타이머(10분) | AUTH-001 |

### 2.1.1 새 탭 / 시작 페이지 (★ NEWTAB-001)

| 화면 ID | 목적 | 진입점 | 핵심 구성 | 관련 기능 |
|---|---|---|---|---|
| #NEWTAB | 브라우저 시작 페이지·새 탭으로 매번 진입하는 hub | 사이드바 "🏡 새 탭" / URL `/` (홈화면 모드 ON 시) / 브라우저 시작 페이지 설정 | (1) 시간 기반 인사("좋은 아침이에요") + 현재 날짜·시각 (2) 중앙 큰 검색바 (웹 / 내 링크 / 통합 모드 chip) (3) **★ 바로가기 타일 (favicon 원형, 최대 8개, "+ 바로가기 추가" 포함)** (4) 빠른 액션(새 링크·할일·전체보기) (5) 오늘 할 일 위젯 (6) 최근 저장 카드 4개 (7) 자주 가는 폴더 chip | NEWTAB-001, NEWTAB-002 |
| #SHORTCUT-PICKER | 바로가기 후보 선택 모달 | #NEWTAB "+" 타일 클릭 | 미고정 링크 목록(최신순), 클릭 시 `pinned=true` 토글 후 모달 닫힘. 최대 8개 도달 시 안내 | NEWTAB-002 |

`manifest.json` 발췌:
```json
{
  "share_target": {
    "action": "/share/receive",
    "method": "GET",
    "params": { "url": "url", "text": "text", "title": "title" }
  }
}
```

## 4. 네비게이션

### 4.1 모바일 (<1024px)

```
┌──────────────────────────┐
│  헤더 (제목 + 우측 액션)  │
├──────────────────────────┤
│                          │
│       메인 콘텐츠         │
│       (단일 컬럼)         │
│                          │
├──────────────────────────┤
│  [홈] [폴더]  +  [검색] [설정]  │  ← 하단 탭바 (FAB 가운데)
└──────────────────────────┘
```

- **하단 탭바** 5개: 홈 / 폴더 / **+** (FAB) / **할일** / 설정 (검색은 헤더 아이콘)
- **시트 슬라이드 업**: #LINK-ADD, #LINK-DETAIL, 모달

### 4.2 데스크톱 (≥1024px)

```
┌────────┬──────────────────────────────────────┐
│        │  헤더 (검색바 + 알림 + 프로필)         │
│  로고  ├──────────────────────────────────────┤
│        │                                      │
│  홈    │                                      │
│  폴더▼ │      메인 그리드                      │
│  ├개인 │      (1024+: 2열 / 1440+: 3열         │
│  └공유 │       / 1920+: 4열)                   │
│  검색  │                                      │
│  설정  │                                      │
│        │                                      │
│ ─────  │                                      │
│  프로필│                                      │
│  로그아│                                      │
│  웃    │                                      │
└────────┴──────────────────────────────────────┘
            ↑ 1920px+ 시 우측 #LINK-DETAIL 패널 추가
```

- **좌측 사이드바** (240px 고정): 로고 / 홈 / 폴더 트리 (개인 + 공유 섹션) / **할일** / 검색 / 설정 / 프로필+로그아웃 (하단)
- **메인**: 그리드 (2~4컬럼)
- **우측 상세 패널** (≥1920px): #LINK-DETAIL을 패널로 (시트 대신)
- **단축키**:
  - `⌘K` / `Ctrl+K` — 검색
  - `N` — 새 링크 추가
  - `J` / `K` — 카드 다음/이전 이동
  - `Enter` — 상세
  - `Del` — 삭제
  - `?` — 단축키 도움말
- **카드 hover**: 우측 상단에 액션 아이콘 노출 (열기·이동·삭제)

### 4.3 헤더 우측 액션 (화면별)

| 화면 | 우측 액션 |
|---|---|

## 5.1 💻 웹 전용 화면·라우팅·기능 인덱스

> 단일 React PWA 코드베이스에서 viewport·환경 분기로만 활성화. 모바일에서는 미노출 또는 대체 UI.

### 화면

| 화면 ID | 분류 | 활성 조건 | 비고 |
|---|---|---|---|
| #NEWTAB | 진입점 | viewport ≥ 1024px 또는 Chrome 확장 newtab override | NEWTAB-001/002 |
| #SHORTCUT-PICKER | 모달 | #NEWTAB 안 "+" 타일 | NEWTAB-002 |
| #SETTINGS-IMPORT | 데이터 이전 | 데스크톱 | IMPORT-001 (모바일은 안내만) |
| #SETTINGS-EXT | 브라우저 통합 | 데스크톱 | 확장 설치 안내·newtab override 토글·즐겨찾기 sync 권한 동의 UI |

### 라우팅 추가

| 경로 | 화면 | 비고 |
|---|---|---|
| `/newtab` | #NEWTAB | Chrome 확장 newtab override 타깃 |

## 6. 컴포넌트 인벤토리 (1차)

> 와이어프레임/디자인 시스템(다음 단계)에서 토큰화. 현 프로토타입에서 확립된 것 기준.

- **카드**: 링크 카드, 폴더 카드
- **시트**: 모바일 슬라이드업 시트 (LINK-ADD, LINK-DETAIL, 설정 모달)
- **사이드 패널**: 데스크톱 우측 상세 패널 (≥1920px)
- **FAB**: + 버튼 (모바일 탭바 가운데)
- **태그 칩**: 인라인 태그 (1~5개)
- **이모지 picker**: 폴더·아바타 선택
- **인증 입력**: 6자리 코드 박스, 비밀번호 인풋(eye 토글)
- **토스트 / 모달 / 확인 다이얼로그**
- **info-banner**: 비밀번호 변경 화면 안내 배너 (이미 프로토타입에 추가)
- **스와치**: 강조색 3종 선택 UI
- **프로그레스**: import 진행률 바
- **Empty State**: 일러스트 + CTA
