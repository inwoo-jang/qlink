| 분기 | 조건 | 처리 |

| 비밀번호 정책 미충족 | 입력 검증 | "8자 이상, 영문+숫자 포함" 안내, 가입 버튼 비활성 |

| 네트워크 오류 | 어느 단계든 | 토스트 "네트워크 연결을 확인해주세요" + 폼 상태 유지 |
| 프로필 자동 생성 실패 | 인증 후 트리거 | 1초 대기 후 1회 재시도 → 실패 시 로그아웃 + 에러 안내 |

|---|---|---|

| 게스트 모드 | 기능 자체 | 동기화 비활성. 가입 시 마이그레이션 모달로 안내 |
